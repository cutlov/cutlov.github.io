Training data loss large

Model bias

Bias too small which make the smallest loss not contain there.

Solution: redesign you model to make it flexible
such as more feature

Optimization issue

This method can’t find the smallest loss

conclusion

  • Compare different model to find this model efficiently.
  • Start from shallower networks(or other model),which are easier to optimize
  • If deeper networks don’t obtain smaller loss on training data, then there is optimization issue.

testing data loss large(overfitting)

  • Less parameter
  • Less features
  • Early stopping
  • Regularization
  • Dropout
